    Is this ticket on hold?

    • Is this an FLVS Course?

      • Enter Time
      • Save ticket
      • You are done until the course hold is removed
      • In 'Work Notes' inform GS that the course is on hold
      • Enter Time
      • Save ticket
      • You are done until the course hold is removed
    • Continue, below.

    Is it an enhancement?

    • Is this an FLVS Course?
      • Change state to Resolved
      • Close code to 'CHD - pending enhancements'
      • Close note will be canned 'enhancement' comment.
      • In Work Notes add canned 'enhancement' comment.
      • Assign to queue 'GS - Global Technical Client Support (Level 1)'
    • Continue, below.

    More information needed from the customer?

    • Is this an FLVS ticket?
      • Change state to “Awaiting User Info”
      • Send canned comment to customer via the "Additional Comments" box
        • Wait for customer to respond
        • Keep trying for 3 days (resend the message each time). If this is a student ticket, make sure you try a phone call at least once
        • Does the customer finally respond?

        • make state 'active', then continue to next section
        • Change ticket state to 'resolved'
        • Change close notes to 'CHD - Working As Designed'
        • In 'Close Notes' enter message to student about why you are closing the ticket
        • Enter time worked
        • Click 'Save'
        • You are done with this ticket
      • Assign ticket to the 'GS - Global Technical Client Support (Level 1)' queue
      • In 'work notes' send canned comment
      • Enter time worked
      • Click save
      • You are done with this ticket until it comes back to you.
    • Continue, below.

    CS/SME approval needed before editing?

    • in work notes, add 'request approval' template
    • Insert time worked
    • Click 'save and assign to CS'
    • You are done until the ticket comes back to you from the CS
    • Continue, below.

    • Open the exam in Master Manager
    • Create a new 'resolution' template for this ticket
    • In your 'Resolution' template, enter the course name and exam name
      Example:
      RESOLUTION:
      Intensive Reading ( master4688 ) - Exam: 01.00 Module One Pretest
    • In your 'Resolution' template, enter your 'action'
      Example:
      ACTION: Q23 - Replace word in answer choice 2
    • In Master manager click the 'modify' button on the exam question
    • For your 'Resolution' template, copy the exact question stem and answer choices from MM into the 'old' and 'new' sections (don't forget to indication the correct answer in bold)
    • On your 'Resolution' template, edit the 'new'. Make sure you mark the changes from the 'old' and 'new' in red.
    • Copy your 'new' changes into the exam in Master Manager.
    • Click 'Submit' in Master Manager to save the edit
    • Wait for the page to refresh, review to make sure the edit looks correct.
    • Close the exam page.
    • In the main MM window, sync development to production.

    • Add 'scheduled' template message to work notes
    • Add CS name to the work notes recipients

    • Enter time and save.

    • Did CS post comment for a further change?
      • Go back to editing, above
      • Remove CS name from work notes recipients

    Is this a Client 5 course?

    • Fill out Client 5 form and copy 'result' message
    • Insert note into work notes stating that Client 5 notification sent (and paste 'result' message)
    • Continue, below.

    Is this an FLVS course?

    • Is this a ticket created by you?
      • Designate close code
      • Paste resolution into close notes
      • Put time in ticket
      • Change state to resolved and 'save' ticket
      • Paste resolution into work notes
      • Extract CS name
      • Designate close code
      • Insert 'Solved' template message into close notes
      • Insert time worked
      • Change ticket state to 'Resolved' and save ticket
    • Change ticket state to 'Active'
    • Paste resolution into work notes
    • Remove your name from 'Assigned to'
    • Change 'Assignment group' to appropriate GS group (usually GS Global Client Notification)
    • Enter time and 'save' ticket

    Is this a child ticket

    • Are there any other child tickets that you need to finish editing?
      • edit them (go back to the top of the edit checklist)
      • You are done
    • From the parent ticket go to the 'related courses' section
    • Verify which courses truly need the edit (you'll have to go into each course and check)
    • 'Check mark' any related courses that need the edit.
    • Go to the top of the page and copy the main ticket number
    • Go back to the bottom of the 'related courses' section.
    • In the 'actions on selected rows' dropdown, choose 'create incidents'
    • Paste in the ticket number and wait for about 3 seconds and click 'update' button
    • After the page updates, go to the 'child incidents' sections of the ticket
    • For each of the child tickets that you have created, you need to change each of their 'short descriptions' to match the actual course of the ticket (the 'short description' is copied from the parent ticket, so it will originally have the wrong course number and version number---you need to change this in each of the child tickets you created)
    • Edit each child ticket
